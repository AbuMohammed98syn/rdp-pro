التثبيت

ابدأ في دقيقتين

01

Backend على الحاسوب

شغّل الخادم Python على حاسوبك

# تثبيت pip install fastapi uvicorn pillow psutil pyautogui # تشغيل python backend/main.py
02

تثبيت APK على الموبايل

فعّل "مصادر غير معروفة" وثبّت التطبيق

# الإعدادات → الأمان مصادر غير معروفة ✓ # ثبّت RDPPro-v4.0.apk
03

الاتصال

أدخل IP الحاسوب أو امسح QR Code

# أدخل في التطبيق: IP: 192.168.1.x Port: 8000 Token: rdppro-secret-2024
04

Wake-on-LAN (اختياري)

لتشغيل الحاسوب وهو نائم

# احصل على MAC Address: Windows: ipconfig /all # أدخله في حقل MAC # اضغط "إيقاظ"
